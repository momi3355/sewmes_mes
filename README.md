# 프로젝트명 : SEWMES

## 개요

생산 작업 중 불량이 생기기 쉬운 의류의 특성상 전체 공정을 체계적으로 추적 · 관리할 수 있는 MES의 도입이 필요하다고 판단하여 의류 생산 MES 개발

> **MES**란 생산계획부터 제품 출하까지 전 과정을 실시간으로 추적하고 관리하여 생산 현장의 효율성을 높이고 품질을 관리하며 생산비용을 절감하기 위한 시스템.
